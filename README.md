# E-commerce Sales Analysis Platform

## Project Description
This project analyzes e-commerce sales data using Python. It includes data cleaning, analysis, and visualization.

## Tools Used
- Python
- Pandas
- SQLite
- Matplotlib

## Features
- Data cleaning (remove missing values and duplicates)
- Data analysis (total sales, average profit)
- Sales analysis by region
- Data storage using SQL database
- Data visualization (bar chart)
