# E-commerce Sales Analysis Platform

## Project Description
This project analyzes e-commerce sales data using Python. It includes data cleaning, analysis, and visualization to extract meaningful insights.

## Tools Used
- Python
- Pandas
- SQLite
- Matplotlib

## Features
- Clean raw data(remove missing values and duplicates)
- Calculate total sales and average profit
- Analyze sales by region
- Save processed data into a database
- Generate visualizations (bar chart)

## How to Run
 1. Make sure you have Python installed
 2. Install required libraries:
      pip install pandas matplotlib
 3. Run the script:
      python clean_data.py

## Project Structure
 • sales.csv → raw dataset
 • cleaned_sales.csv → cleaned dataset
 • clean_data.py → main script
 • sales.db → database file
 • sales_chart.png → visualization output

## Output Example 
The project generates a bar chart showing sales distribution by region.
